Git Tutorial

When you start a project:

  1. git init
  2. git add -A
  3. git commit -m"start of my css tutorial"
  4. git remote add origin URL
  5. git push origin main or master

When you fork a project

  1. Fork and run git clone
  2. git add -A
  3. git commit -m"start of my css tutorial"
  4. git push origin main or master
